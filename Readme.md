# SimonCompassHelper

## Interface: 20505
## Title: SimonCompassHelper
## Version: 1.0.0
## Author: andreassteffensen
## Notes: A helper addon for the Simon Compass puzzle in World of Warcraft.
## X-Website: https://github.com/andreassteffensen/SimonCompassHelper
## License: MIT
## Description:
SimonCompassHelper assists players with the Simon Compass puzzle by providing helpful hints and tracking puzzle progress.

---

### Features
- Tracks Simon Compass puzzle steps
- Provides visual and/or audio cues
- Lightweight and easy to use

---

### Installation
1. Download the latest release from [CurseForge](https://www.curseforge.com/wow/addons/simoncompasshelper) or [GitHub](https://github.com/andreassteffensen/SimonCompassHelper/releases).
2. Extract the folder into your `World of Warcraft/_retail_/Interface/AddOns/` directory.
3. Restart WoW or reload your UI.
